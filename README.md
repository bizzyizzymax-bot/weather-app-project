#Weather App Project
Here I architected a weather app project where I uses the OpenWeather API using an API key and implemeneted into my Javascript module.

## Live Demo:
https://bizzyizzymax-bot.github.io/weather-app-project/

## Features:
- A search bar and button that allows the user to search a city's weather
- Stylish background video based on the city's weather

## Tech Stack:
-Javascript, HTML5, CSS, OpenWeather API

## What I learned:
-How to use async and promise functions
-How to try a promise and throw and error if the requested information fails
